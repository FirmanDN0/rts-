# **Website Requirement — Rencana Tuhan Studio (RTS)**

## **Konsep Utama**

Website RTS dibuat sebagai **website profesional creative studio / production house** yang menggabungkan:

* Portfolio studio  
* Showcase karya  
* Informasi layanan  
* Sistem konsultasi harga otomatis  
* Manajemen client melalui dashboard admin

Referensi utama:

### **Design Style:**

Mengikuti style **Granator Film**

* Minimalis  
* Premium  
* Clean  
* Fokus pada visual karya  
* Tidak terlalu banyak elemen  
* Banyak ruang kosong (whitespace)  
* Modern creative agency feel

### **Feature Inspiration:**

Mengambil konsep dari **Botbox Production**

* Portfolio showcase  
* Service section  
* CTA untuk konsultasi  
* Sistem request project  
* Tampilan profesional production house

---

# **ROLE 1 — USER / CLIENT**

## **1\. Landing Page**

Tampilan awal harus memberikan kesan:

"RTS adalah studio kreatif yang profesional dan terpercaya"

Isi halaman:

### **Hero Section**

Visual utama:

* Video/showreel  
* Foto project terbaik  
* Animasi ringan

Text:

"Creating Stories Through Visual"

Button:

* View Project  
* Start Project

---

# **2\. Portfolio / Project Showcase**

User dapat melihat berbagai hasil project RTS.

Tampilan mengikuti production house:

Setiap project memiliki:

* Thumbnail/video  
* Judul project  
* Kategori  
* Tahun  
* Deskripsi singkat

Kategori:

* Film Production  
* Animation  
* Motion Graphic  
* Creative Visual

Style:  
Grid portfolio seperti website agency.

---

# **3\. Latest Offer / Update**

Section untuk menampilkan:

* Promo layanan  
* Paket terbaru  
* Pengumuman RTS

Contoh:

"Open Collaboration Project 2026"

Admin dapat mengubah konten.

---

# **4\. Service Section**

Menampilkan 3 layanan utama RTS.

Setiap layanan:

* Nama layanan  
* Foto  
* Deskripsi singkat  
* Tombol konsultasi

Contoh:

### **Film Production**

Membantu produksi film mulai dari konsep hingga hasil akhir.

### **Animation**

Pembuatan animasi 2D/3D untuk kebutuhan kreatif.

### **Creative Visual**

Motion graphic, branding visual, dan kebutuhan digital.

---

# **5\. Sistem Konsultasi Harga**

Fitur utama website.

User dapat melakukan estimasi harga layanan.

Form input:

## **Data Client**

* Nama  
* Nomor WhatsApp  
* Email (optional)

## **Kebutuhan Project**

Jenis layanan:

* Film  
* Animation  
* Motion Graphic

Durasi pengerjaan:

* Cepat  
* Normal  
* Kompleks

Lokasi:

* Studio RTS  
* Lokasi client  
* Luar kota

Tambahan:

* Talent  
* Equipment  
* Request khusus

---

# **Automatic Price Calculator**

Sistem menghitung estimasi harga.

Output:

Contoh:

Estimasi biaya:

**Rp 5.000.000 — Rp 8.000.000**

Catatan:

"Harga bersifat estimasi dan masih dapat disesuaikan melalui konsultasi."

---

# **6\. WhatsApp Negotiation**

Setelah hasil estimasi muncul:

Button:

Format pesan:

---

Halo RTS, saya ingin konsultasi project.

Layanan:  
Animation

Durasi:  
1 bulan

Lokasi:  
Surabaya

Estimasi:  
Rp 5.000.000 \- Rp 8.000.000

---

User tidak perlu mengetik manual.

# **Design Requirement**

## **Visual Style**

* Minimal
